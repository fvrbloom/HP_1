## Описание практической работы

В этой практической работе я изучала основные операции с массивами и динамической памятью в языке программирования C++, а также анализировала различия между последовательными и параллельными алгоритмами с использованием технологии OpenMP.

Моя работа состоит из трёх заданий:
- работа с массивами и поиск минимального и максимального значений
- изучение базовых структур данных и операций над ними
- работа с динамической памятью и вычисление среднего значения элементов массива с использованием последовательного и параллельного подходов

## Задание 1 — Массивы: Min/Max

### Цель

Я создала массив целых чисел и реализовала алгоритмы для поиска минимального и максимального значений, используя как последовательный, так и параллельный подходы.

### Описание алгоритма

1. Пользователь вводит размер массива `N`
2. Я динамически выделяю память для массива
3. Массив заполняется случайными целыми числами в диапазоне от 1 до 100
4. Я использую последовательный алгоритм для поиска минимального и максимального значений
5. Я измеряю время выполнения последовательного алгоритма
6. Я применяю параллельный алгоритм с использованием OpenMP и механизма `reduction` для поиска минимального и максимального значений
7. Я измеряю время выполнения параллельного алгоритма
8. Я сравниваю результаты и время выполнения

### Используемые технологии

- Динамические массивы (`new[]`, `delete[]`)
- Последовательные алгоритмы
- Параллелизация с OpenMP (`#pragma omp parallel for`)
- Операции редукции (`min`, `max`)
- Измерение времени с помощью `std::chrono`

## Задание 3 — Динамическая память и среднее значение

### Цель

Я вычислила среднее значение элементов массива, используя динамическое выделение памяти, и сравнила последовательную и параллельную реализации.

### Описание алгоритма

1. Пользователь вводит размер массива `N`
2. Я динамически выделяю память для массива
3. Массив заполняется случайными целыми числами
4. Я вычисляю среднее значение последовательно
5. Я измеряю время выполнения последовательного алгоритма
6. Я вычисляю среднее значение параллельно, используя OpenMP и механизм `reduction`
7. Я измеряю время выполнения параллельного алгоритма
8. Я сравниваю результаты и время выполнения, а также вычисляю ускорение

### Используемые технологии

- Динамическое выделение памяти
- Последовательные вычисления
- Параллельные вычисления с OpenMP
- Операция редукции (`+`)
- Измерение времени с помощью `std::chrono`
